# nc -u -s 127.0.0.1 -p 12345 -w 1 127.0.0.1 9999  with stdin = "hello".
# nc binds the source to 127.0.0.1:12345 (so the expect's `from` is
# deterministic; without -s we'd get 0.0.0.0 which doesn't exact-match) and
# sends a single UDP datagram. -w 1 caps the post-send wait so the test
# doesn't dawdle.
expect from=127.0.0.1:12345 to=127.0.0.1:9999 hex=68656c6c6f
